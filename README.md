# 🚀 ConnectHub - MERN Social Media Platform

ConnectHub is a modern full-stack social media application built using the MERN Stack. Users can create posts, interact with content, manage profiles, and connect with other users.

---

## ✨ Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

### Posts
- Create Post
- View Feed
- Like Posts

### Upcoming Features
- Comment System
- Follow / Unfollow Users
- User Profiles
- Media Uploads
- Real-Time Messaging
- Notifications

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcryptjs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/nitinkarma04-afk/ConnectHub-MERN.git
```

### Install Dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

### Run Backend

```bash
npm run dev
```

### Run Frontend

```bash
npm run dev
```
